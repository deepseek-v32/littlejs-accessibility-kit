Integration with LittleJS

This kit works alongside LittleJS's existing systems:

  • Uses LittleJS input system for keyboard/gamepad
  • Integrates with LittleJS audio controls
  • Complements LittleJS rendering pipeline
  • Adds accessibility without breaking existing games
